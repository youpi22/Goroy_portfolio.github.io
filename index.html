Unity Game Project

PAMANA is an Android game that immerses players in Philippine culture through storytelling and interactive gameplay. With five levels, each featuring mini-games like puzzles, quizzes, and memory challenges, players explore maps, collect objects within a time limit, and interact with NPCs who guide them. Using Dijkstra's algorithm for navigation, the game combines fun and education to celebrate and share the rich heritage of the Philippines.
